# ViralForge AI 🚀

AI-powered viral video automation SaaS platform.

## Features
- AI Script Generator
- AI Voiceover
- Auto Captions & Subtitles
- Hashtag Generator
- Thumbnail AI
- Social Media Auto Upload
- No Watermark Export

## Tech Stack
- Next.js
- FastAPI
- PostgreSQL
- FFmpeg
- Gemini / OpenAI API

## Setup Guide

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
